Składnia stylu zewnętrznego

Opis stylu zewnętrznego:

Styl zewnętrzny jest wstawiany w sekcji head dokumentu HTML za pomocą znacznika link. Może on wystąpić wie
razy w dokumencie, aby dołączyć kilka różnych arkuszy stylów.
Możliwe jest dołączenie dowolnej ilości arkuszy stylów do jednego dokumentu HTML.
W przypadku konfliktów, przeglądarka będzie stosować style z ostatniego arkusza stylów, który został
załadowany.
Pliki arkuszy stylów mogą być tworzone i edytowane za pomocą zwykłego edytora tekstowego takiego jak Notat
lub Sublime Text.
Istnieją specjalne edytory CSS, takie jak Adobe Dreamweaver, CSS Designer czy Visual Studio Code, które
ułatwiają tworzenie i edytowanie.
Pliki będące zewnętrznymi arkuszami styli posiadają rozszerzenie .css
Umożliwia on oddzielenie kodu HTML i CSS, co poprawia czytelność i ułatwia utrzymanie kodu.
Możliwość łatwego dzielenia i łączenia arkuszy stylów między różnymi dokumentami HTML, co pozwala na użyci
tych samych stylów na wielu stronach bez konieczności ich powielania.

Uwagi na temat pisania w CSS

  • Język CSS dopuszcza stosowanie zarówno dużych, jak i małych liter. Poprawnymi nazwami atrybutu są background, backGROUND oraz BACKground.
  • Wstawienie białych znaków (spacje, tabulacje, znaki nowej linii) w arkuszu CSS nie ma wpływu na jego działanie.
  • Dlatego możesz ułożyć wpisywane reguły CSS w taki sposób, aby były bardziej czytelne
  • Komentarze w arkuszu stylów pisze się używając znaków: /* (otworzenie komentarza) oraz */ (zamknięcie komentarza), np.: /*to jest komentarz*/. Wszystko pomiędzy tymi znakami jest ignorowane przez przeglądarkę.
